# ChatGPT Double Enter Send

A minimal Chrome extension for ChatGPT editing mode.

## Feature

- `Enter` while editing a message inserts a newline.
- `Shift + Enter` inserts a newline.
- Press `Enter` twice quickly to submit the edited message.

All previous bookmark, sidebar, settings, export, theme, language, storage sync, and context menu features have been removed.

## Installation

1. Open Chrome and go to `chrome://extensions`.
2. Enable Developer mode.
3. Click Load unpacked.
4. Select this project folder.
5. Open or refresh [ChatGPT](https://chatgpt.com/).
